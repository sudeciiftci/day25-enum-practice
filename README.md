# Java Enum Practice

This project demonstrates the use of **enum types** in Java.

## Concepts

### Basic Enum
- Fixed set of constants
- Example: `Season`, `UserRole`

### Looping Enums
- `values()` method returns all enum constants

### Enum with Fields
- Enum can have variables, constructors, and methods
- Example: `Difficulty` with points

## Examples

### Season
- SPRING, SUMMER, AUTUMN, WINTER

### UserRole
- ADMIN, USER, MODERATOR

### Difficulty
- EASY = 10 points
- MEDIUM = 20 points
- HARD = 30 points

## Purpose
- Understand enum usage
- Store fixed constant values safely
- Add extra data to enums
